# My Journey of Creating a Personal Blog Website

Creating a personal blog website was an exciting and educational experience for me. As a Computer Engineering student, this task helped me understand how web technologies work in real-life applications.

## 🚀 Introduction

I am Ahmad Talib, currently pursuing a degree in Computer Engineering. I completed my Matric with **93%** and Intermediate with **87%**. I have a strong interest in programming and technology, and Python was my first programming language.

This blog is part of my academic lab task, but it also represents my first step towards building my online presence.

## 💡 Objectives of This Project

The main goals of this project were:
- To learn how to create a personal website using GitHub Pages
- To understand the structure of static websites
- To gain hands-on experience with Jekyll
- To improve my web development skills

## 🛠️ Tools and Technologies Used

In this project, I used:
- GitHub Pages for hosting the website
- Jekyll as a static site generator
- Markdown for writing content
- Basic HTML and CSS for design customization

## 📚 What I Learned

Through this project, I learned many important concepts:

### 1. Static Website Development
I understood how static websites are created and hosted without using databases or backend servers.

### 2. Version Control with GitHub
I learned how to manage files, commit changes, and publish a live website using GitHub.

### 3. Jekyll Structure
I explored how Jekyll works, including layouts, posts, and configuration files.

### 4. Writing in Markdown
Markdown made it easy to format content in a clean and readable way.

## 🌐 Features of My Website

My website includes:
- A Home page with my introduction
- An About page describing my background
- A Contact page for communication
- A Blog section where I share my experiences

## 🎯 Conclusion

This project was a valuable learning experience for me. It not only improved my technical skills but also gave me confidence in building and managing a website.

In the future, I plan to enhance my website by adding more features, improving design, and sharing more blog posts about my learning journey.
